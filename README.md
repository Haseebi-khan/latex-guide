# LaTeX
This repository is a complete learning and reference hub for LaTeX. It includes:

- Basic document structure (documentclass, packages, page setup)

- Text formatting (fonts, spacing, lists, paragraphs)

- Mathematical expressions (inline, display, equations, symbols)

- Tables & figures (tabular, longtable, images, captions)

- Environments (enumerate, itemize, custom environments)

- Common LaTeX errors and their fixes

- Templates for assignments and academic documents

- Cheatsheets and quick examples

Designed for students and learners who want a simple, practical, and error-free introduction to LaTeX.
